1. Select Operating Folders

Select the source folder that will be scanned recursively for the types of media files you specify.

Select the destination folder that files will be sorted into.

2. Select Media Types

Select the types of files you want to analyze.

3. Select Sort Parameters

Select the parameters used to group media files, like size, bitrate, etc.

4. Analyze and Preview

Scan for the selected files and display the proposed sorting results.

5. Apply Sorting

Apply the proposed changes to the filesystem. THIS CANNOT BE UNDONE!
